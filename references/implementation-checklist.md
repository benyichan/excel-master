# 交付前验证清单

## 格式完整性

- [ ] A 列若含数据，表头已应用蓝底白字右对齐、数据 Arial 11 左对齐、合计行浅蓝底粗体
- [ ] 边框循环的 `border_start` 正确：A1有值时从 1 开始，否则从 start_col 开始
- [ ] 所有单元格都统一 Font(name='Arial', size=11)，数字列叠加颜色（蓝/黑），文本列保持默认黑
- [ ] 行高 18 已应用
- [ ] 隐藏网格线已设置
- [ ] 千分位符格式正确（money→#,##0.00, number→#,##0, pct→0.00%）
- [ ] 右侧空白列（宽 3）已添加
- [ ] 冻结首行已设置

## beautify 专项

- [ ] 公式保留：所有 `=` 开头的单元格未被转值
- [ ] 公式颜色：公式→黑色，值→蓝色（双重检测 `str.startswith('=')` + `cell.data_type == 'f'`）
- [ ] A 列自适应宽度（空→2，有数据→min10 max50）
- [ ] `_detect_data_range()` 正确识别表头行（A 列全空时确认首行确实是表头）

## 数据逻辑

- [ ] 合计 = 明细之和
- [ ] 占比 = 部分/整体 × 100%
- [ ] 示例表数据经得起推演

## 主题校验
